# RGB_Run

SET09121 Games Engineering - Bohdaniec & Morel-Barnes

How to run : 
1. add a lib/sfml with the most recent sfml version from git. 
2. Generate the build using CMake. 
3. enjoy. 

Controls: 
Original controls are not shown so here they are : 
Q W E - switch colors 
space - fire 
up - Jump 
left right - Move.
